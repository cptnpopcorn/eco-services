# Services environment variables

Create a file `services.env` containing something like

[source, shell]
----
DOMAIN=my.domain.org
MARIADB_TZ=Europe/Zurich
MARIADB_ROOT_PASSWORD=12345spaceballs
LB_DB_USER_PWD=12345spaceballs
LB_INSTALL_PWD=12345spaceballs
BAM_WEB_STATIC_USR=spaceman
BAM_WEB_STATIC_PWD=12345spaceballs
----

LB_DB_USER_PWD:: the username to be used by librebooking
LB_INSTALL_PWD:: the admin installation password for initial librebooking setup
BAM_WEB_STATIC_USR:: SSH user for updating static web content for 'bam' subdomain via SFTP
BAM_WEB_STATIC_PWD:: SHH user password for 'bam' SFTP
